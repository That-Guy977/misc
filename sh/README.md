### Execution

- CLI
  - `. script.sh` - Execute `.sh` file
  - `sh script.sh` - Execute `.sh` file
  - `bash script.sh` - Execute `.sh` file with `bash`
  - `sh` - Shell REPL
  - `bash` - Bash REPL
